# ♟️ Smart Chess Hub

Шахматная платформа с ИИ противником, авторизацией и историей партий.

## Для кого

Для всех кто хочет тренироваться в шахматы и следить за прогрессом.

## Что сделано

- Полные правила шахмат (рокировка, взятие на проходе, мат, пат)

- Игра против Stockfish AI (3 уровня сложности)

- Авторизация через Supabase

- История игр и профиль игрока

- Тёмная и светлая тема

- Адаптивный дизайн для телефона

## Стек

React 19, TanStack Router, Supabase, chess.js, react-chessboard, Stockfish, Tailwind CSS
