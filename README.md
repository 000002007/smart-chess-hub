# ♟️ Smart Chess Hub

Шахматная платформа нового поколения с ИИ-тренером и рейтинговой системой.

## Для кого

Для всех, кто хочет улучшать игру, отслеживать прогресс и соревноваться с игроками по всему миру.

## Что сделано

- Полные правила шахмат (рокировка, взятие на проходе, мат/пат)
- Игра против Stockfish AI (Easy/Medium/Hard)
- Авторизация и профиль игрока с рейтингом ELO
- История всех партий с PGN-записью
- AI Coach — анализ партии после игры
- Глобальный лидерборд с фильтром по городам
- Тёмная/светлая тема, адаптивный дизайн
- Монетизация: кнопка Upgrade to Pro

## Стек

React 19, TanStack Router, Supabase, chess.js, Stockfish.js, Tailwind CSS
